# eslint-config-treemmett

## Installation

```bash
# with yarn
yarn add -D eslint eslint-config-treemmett

# or if you prefer npm
npm i -D eslint eslint-config-treemmett
```

## Usage

Extend your eslint config

```json
{
  "extends": [
    "treemmett",
    "treemmett/cypress",
    "treemmett/jest",
    "treemmett/node",
    "treemmett/react"
  ]
}
```
